# 내 프로그래밍 세계에 오신 것을 환영합니다!

내 GitHub 프로필에 오신 것을 환영합니다. 2021년에 프로그래밍 여정을 시작하여 이후 계속해서 기술을 연마해 왔습니다. 현재 Junior Software Developer로 일하고 있으며 학습, 창작, 개발, 문제 해결에 대한 열정으로 일합니다. 이 프로필은 공개 저장소에서의 작업과 내 성장에 큰 도움이 되는 별표가 있는 저장소를 보여줍니다. 협업이나 기술 관련 토론을 위해 연결하고 탐색하는 것을 주저하지 마십시오.
